## Getting Started

### Dependencies
  - tcod 
  - numpy

### Controls 
  - Arrowkeys for up, down, left, right
  - v for message view
  - i for inventory
  - g for pickup item
  - d for drop item
  - period + shift for descend stairs
  - escape to exit
 
To use an item:
    Open inventory with i, use letter listed to consume
